Welcome to Telemetry System For RC Car!

This project mainly focuses on wirelessly transmitting the details of a remotely controlled vehicle to the ground station with the help of radio waves. Our project offers you two categories of Visualization for the incoming data. Go ahead and try them!

This section visualizes various properties related to the vehicle namely, Velocity, Temperature, Orientation, GPS Location and Battery Percentage.

With the help of CSS, HTML, JavaScript and Python Libraries, this section displays the live data in the form of graphs.
