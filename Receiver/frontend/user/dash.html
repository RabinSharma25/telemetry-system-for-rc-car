Welcome to Telemetry For RC Car!

Our project offers you two categories of Visualization for the incoming data from a remote vehicle.

Go ahead and try them!

Telemetry Visualization

With the help of CSS, HTML and JavaScript, this section visualizes various properties related to the vehicle namely, Velocity, Temperature, Orientation, GPS Location and Battery Percentage of the batteries in the vehicle.

Live Graphs

With the help of CSS, HTML, JavaScript and Python Libraries, this section displays the data in the form of live graphs using various technologies.
